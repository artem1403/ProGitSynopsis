# ProGitSynopsis
Конспект и основные команды, собранные при прочтении книги "Pro Git" 2-nd edition, 2014. Информацию собираю в личных целях, как краткое пособие и шпаргалка.

### Часть 1 - базовые команды.
* создать репозиторий
* добавить в индекс
* коммит
* отправка на сервер
* получение с сервера
* просмотр и создание тегов
* просмотр дифа
